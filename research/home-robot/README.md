# Home Robot Project — Research Folder

A connected, spatially-aware home robot platform. Single mobile unit (with optional fixed observers) that hears you, sees you, knows where things are, can be sent on errands, and answers questions grounded in what it has observed.

## What this folder contains

| Folder | Contents |

| `01-vision/` | What we're building, why, who it's for |
| `02-architecture/` | System architecture, capability layers, contracts |
| `03-hardware/` | BOM tiers, sourcing, hardware decisions |
| `04-software/` | Software stack, agent design, world model |
| `05-reliability/` | Docking, charging, always-on engineering |
| `06-reviews/` | Reviews from CTO, researcher, consultant, users, chief of staff |
